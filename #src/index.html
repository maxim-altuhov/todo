Список дел

  • Для добавления задачи в список нужно написать текст в поле ввода и нажать Enter или значек "+"
  • Чтобы удалить задачу нажмите на корзину напротив этой задачи
  • Чтобы вычеркнуть задачу нажмите левой кнопкой мыши на саму задачу
  • Нажмите "Сохранить", чтобы сохранить список задач на потом
  • Нажмите "Очистить", чтобы полностью удалить список задач
